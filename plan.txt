-------  Введение ----------------

Всем, добрый вечер, представляю результат работы над учебным проектом,
целью которого было создание фронтенд-части интернет-магазина по продаже товаров для животных.

Приложение создано на основе фреймворка РЕАКТ.
Также в приложении использованы дополнительные библиотеки:

        "axios": "^1.10.0" -                для http  запросов на бекенд,

        "redux": "^5.0.1",
        "@reduxjs/toolkit": "^2.8.2",
        "react-redux": "^9.2.0",
        "redux-persist": "^6.0.0",          для управления глобальным состоянием приложения

        "react-hook-form": "^7.60.0",
        "@hookform/resolvers": "^5.1.1",
        "yup": "^1.6.1"                     для управления и валидации форм
        
        "react-router-dom": "^7.7.0",       для навигации по приложению
        
        "react-select": "^5.10.2",          компонент-аналог html тегу SELECT с возможностью стилизации - применяется в панели фильтров и сортировки 
        
Приложение доступно по адресу     https://pet-shop-frontend-blue.vercel.app/    
        
------  Функциональность -----------------

HEADER
    логотип     - переход на главную
    меню        - переход на соотвествующие страницы
    корзина     - переход на страницу корзина
                    при наличии товаров в корзине, отображается бейдж с количеством товаров


FOOTER
    секция КОНТАКТЫ -   телефон
                        местоположение
                        иконки соц.сетей
    секция карты        гулмепс

Гланая страница
    Баннер с кнопкой перехода на страницу с акционными товарами
    Секция  ПОПУЛЯРНЫЕ категории            - переход на страницу товаров данной категории
    Секция  ПОПУЛЯРНЫЕ товары со скидкой    - переход на страницу с информацией о товаре
                                            - кнопка добавления товаров в корзину   -   если товар есть в корзине, стиль кнопки меняется  
    Баннер с формой запроса скидки на первый товар  - при отправке вместо формы сообщение      
    Заголовки с кнопкой перехода на соответствующую страницу

Страница всех категорий
    карточки категорий            - переход на страницу товаров данной категории
    Breadcrumbs

Страница всех товаров
    карточки товаров                - переход на страницу с информацией о товаре
                                    - кнопка добавления товаров в корзину   -   если товар есть в корзине, стиль кнопки меняется    
    панель фильрации и сортировки
